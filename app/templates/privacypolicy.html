Privacy Policy

Last updated: January 2026

SpecDrift (“we”, “our”, “us”) respects your privacy. This Privacy Policy explains what information we collect, how it is used, and how it is protected when you use the SpecDrift website and services.

1. Overview

SpecDrift is an API contract comparison tool designed for developers. Our guiding principle is simple:

We do not collect or store your API specifications or personal data unless absolutely necessary.

2. Information We Do NOT Collect

  • User accounts or authentication data
  • Uploaded OpenAPI specifications
  • Stored comparison results
  • Personal identifiers
  • User tracking across sessions

Uploaded files are processed in memory only and discarded immediately after the comparison is completed.

3. Information We May Collect

a. Technical & Usage Data

We may collect limited, anonymous technical information such as:

  • Browser type
  • Operating system
  • Pages visited
  • General usage statistics

This data is used solely to improve performance, understand usage patterns, and maintain service reliability. It cannot be used to personally identify you.

b. Server Logs

Server logs may temporarily record request metadata such as timestamps, HTTP status codes, and error messages. These logs do not include uploaded API specifications and are periodically rotated or deleted.

4. Cookies

SpecDrift does not use tracking cookies. If analytics are enabled, they are privacy-focused, cookie-less, and used only for aggregated metrics.

5. Third-Party Services

We may use third-party infrastructure providers (such as hosting or analytics services) strictly to operate the platform. These providers do not receive your uploaded API specifications and are required to adhere to appropriate data protection practices.

6. Data Security

We use reasonable technical and organizational safeguards to protect the service, including in-memory processing and secure server configurations. However, no method of transmission over the internet is completely secure.

7. Children’s Privacy

SpecDrift is intended for developers and professionals. We do not knowingly collect personal information from children under the age of 13.

8. Changes to This Policy

We may update this Privacy Policy from time to time. Any changes will be posted on this page with an updated revision date.

Plain-English Summary:
No accounts. No stored API specs. No selling data. No creepy tracking. Just a tool that does one job well.
